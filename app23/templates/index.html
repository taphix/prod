 Description of the Set

Quality craftsmanship

The Aviator tool kit includes a wide range of the most in-demand tools that will help you cope with any task. The set includes:

- Keys: A set of keys of various sizes for working with any fasteners.

- Screwdrivers: A set of screwdrivers with different attachments, including cross and flat ones.

- Pliers: Universal pliers for grabbing, holding and cutting wires.

- Hammer: A durable and balanced hammer for hammering nails and performing other work.

- Roulette: An accurate roulette for measuring distances and objects.

- Hacksaw: A reliable hacksaw for cutting wood, plastic and other materials.
